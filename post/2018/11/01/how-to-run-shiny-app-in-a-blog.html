How to run shiny app in a blog

Kazu Nozue

Tags: R; shiny; reproducibility

Shiny is an interactive application written in R using Shiny package. A Shiny app can be run on local RStudio or on server. If a Shyny app’s URL is available, you can embed the URL into <iframe></iframe> (a HTML tag). An output is html by following instruction. Also this blog is made by markdown and posted by blogdown R package that uses Hugo language for static website framework.

Instruction

step1. Click Preview. step2. In the preview opened, click “Open in Browser”.

Note that to visualize code, add four spaces or one tab at the beginning. See Markdown guide.
